# 🐙 OctoScout

**LLM-Powered GitHub Issues Agent: Search, Ask, and Give Back**

Ever spent hours digging through thousands of GitHub Issues to debug a version incompatibility? OctoScout does that for you.

OctoScout is an AI agent that understands your code and environment, searches GitHub Issues with version-aware context, and helps you give back to the community when your problem gets solved.

## What It Does

- 🔍 **Search** — Find solutions from GitHub Issues/Discussions/PRs, with version-aware filtering and validation
- ❓ **Ask** — Auto-draft high-quality issues when no solution exists
- 🤝 **Give Back** — Found someone with the same unsolved problem? One click to share your solution
